# 中場提醒  

1. 老師提醒引入套件的 script 的時候盡量放在 ``</body>`` 的上面    
2. 因為程式是一行一行讀的, 所以引入的時候要注意   
3. CSS 命名的時候多用 `-`, JS 用小駝峰  
4. 變數的作用域要多注意
