# GitHub 補充 - 靜態頁面   

- 程式碼管理也是時程管理的一環  
- 靜態網頁 → 網頁本身沒有變動, 一樣可以有 js 
    靜態頁面可以做動態的, 不過資料儲存在前端, 只是所有東西會被看光, 但可以做變動的東西  
- 動態網頁 → 有跟資料庫串接的方式, 同步/非同步  

==要作靜態頁面的話檔案一定要是 index.html==
