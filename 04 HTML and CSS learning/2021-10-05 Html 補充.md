# HTML 5 標準之前的補充   

![](https://i.imgur.com/Ni0z5mU.png)

- 以前前端跟後端相依性很強   
> 壞處是今天加新功能, 常常要去後端加   
> 可能用打洞的方式去從後端拿, 相依性高   


- 現在前後端分離   
> 用 API 分離  
> 藉由表單, 資料非同步, 去跟 Server 要資料  
> 好處是如果要增加東西, 在前端下表單, 送到後端去拿資料  
> 比較能減緩兩邊相依性, 不用去互相等
